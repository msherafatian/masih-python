# MASIH-Python

**Modular Analysis Suite for Interactive Heterogeneity**

A Python/Dash implementation of MASIH for interactive single-cell RNA-seq analysis.

## Overview

MASIH-Python provides a user-friendly web interface for comprehensive single-cell RNA-seq analysis, including:
- Quality control and preprocessing
- Clustering analysis
- Marker gene detection
- CancerSEA pathway scoring
- Trajectory inference
- Interactive visualization and export

This is the Python implementation of MASIH, maintaining architectural parity with the [R/Shiny version](https://github.com/msherafatian/masih).
